Quem sou eu?

Meu nome é Gustavo Henrique Caetano. Sou um estudante de Sistemas de Informação. Planejo seguir carreira acadêmica e científica como Engenheiro de Software. Mestrado, Doutorado e PhD estão definitivamente nos planos. Meu objetivo primário de carreira é "aprender, dominar e implementar", portanto, pretendo me dedicar em certos temas de interesse e domina-los visando trabalhar em um laboratório participando de uma pesquisa ou produto de uma grande empresa, organização ou startup com minha área de conhecimento e ser reconhecido pelo bom trabalho.

Sistemas Operacionais

Eu uso Windows. Particularmente, eu admiro muito o MacOS da Apple, pelo seu design e também por gostar muito de sistemas baseados no UNIX e o MacOS por ser mais famoso (que GNU/Linux) possui uma variedade de programas ainda maior, mas já que estamos falando de Brasil e tudo aqui é muito caro, eu uso e gosto muito de distribuições Linux, que também são sistemas baseados em UNIX. Gosto de muitas distribuições e já usei várias, inclusive costumo acompanhar o site Distrowatch para ficar ligado no Ranking de popularidade e notícias/novidades das distribuições, mas hoje em dia uso especificamente o Windows 10.

O Que Faço Em Meu Computador?

Eu passo a maior parte do meu tempo lendo ou assistindo vídeos/tutoriais de computação, matemática e jogos, sobre algum tema de interesse. O navegador Web que utilizo é principalmente o Microsoft Edge, mas já que também desenvolvo para Web e eu sempre quero ter certeza de que as pessoas são capazes de usarem meus projetos, costumo testar os projetos no Firefox e no Chrome também, então mantenho os dois atualizados. Quando não estou fazendo isso, provavelmente estou fazendo algum código de teste sobre algum conceito novo que aprendi ou pensando em melhorias ou ideias de projetos. Eu uso VS Code como editor de texto para Python, HTML, CSS e JavaScript; Intellij IDEA e Netbeans como IDEs para Java; CodeBlocks e Geany como IDEs para C e C++.

Linguagens de Programação

Tenho uma tendência a gostar de linguagens orientadas a objetos, programação orientada a objetos é um paradigma que vem natural para mim, então eu gosto muito de Python, Java e C++. A escolha da linguagem depende do tipo de projeto de software que tenho em mente, na maior parte do tempo fico pensando em sistemas Web e para isso tenho Python e Flask como ferramentas e por se tratar de Web, HTML e CSS são incluídos, mesmo não sendo linguagens de programação e sim linguagens de marcação. Por conveniência eu não faço HTML e CSS "from scratch", ou seja, "do zero", eu uso Bootstrap Framework como utensílio para agilizar o design, até porque, dessa forma acredito que posso fazer coisas maiores, melhores, bonitas e responsivas. Se for um sistema para Desktop de grande porte provavelmente irei arquitetar e implementar em Java e usar JavaFX se uma interface gráfica for necessária. Para estudos de coisas mais simples, como algoritmos e soluções de problemas costumo usar Python ou C++.
